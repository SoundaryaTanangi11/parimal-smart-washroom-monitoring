# 🚀 Parimal - Smart Washroom Monitoring System

## 📌 Overview
Parimal is a Smart Washroom Monitoring System designed to track cleanliness and environmental conditions in real-time.

It provides an interactive dashboard for monitoring:
- Temperature
- Humidity
- Gas Levels
- Sound Levels

---

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL (WAMP)
- Charts: Chart.js

---

## 🔐 Features
- Secure Admin Login
- Interactive Dashboard with Cards & Charts
- Sensor Data Visualization
- Location & Device Tracking
- Product Details Management
- Demo Mode with Simulated Data

---

- Glassmorphism UI Design
